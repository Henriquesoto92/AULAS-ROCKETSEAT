# Flexbox

* Nos permite posicionar os elementos dentroda caixa
* Controle em uma dimensão (horizontal ou vertical)
* Alinhamento, direcionamento, ordenar e tamanhos

```css
div.parent {
    display: flex;
}
````

## flex-direction

* Qual a direção do flex: horizontal ou vertical
* row | column

# Alinhamento

* justify-content
* align-items
